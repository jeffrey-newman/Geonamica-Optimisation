Specify maps that will saved as image files, that are produced when running the Geoproject with the specified saving logging file.

 

Format:

 

For a category map (int representation):

CATEGORISED:LEGEND=”Path-to-legend-file-relative-to-geoproject-directory”:PATH=:”Path-to-map-relative-to-geoproject-directory”:YEARS(XXYY;XXZZ;etc.):DIFF=:”Path-to-differencing-map-relative-to-geoproject-directory”:SAVE_AS=”filename-of-output-map”

 

For a continuous map (double representation):

LINEAR_GRADIENT:LEGEND=”Path-to-legend-file-relative-to-geoproject-directory”:PATH=:”Path-to-map-relative-to-geoproject-directory”:YEARS(XXYY;XXZZ;etc.):DIFF=”Path-to-differencing-map-relative-to-geoproject-directory”:SAVE_AS=”filename-of-output-map”

 

 

NOTES:

1.     include the quotations in the paths to the files.

2.     The SAVE_AS file name is given without the file extension.

3.     Only pngs supported as output file format at this time

4.     if the time-stamp details at the end of a map path are not included, then a series of maps from the years as specified in YEARS() will be used. The years in YEARS() will not be used if a complete (relative) path to a map is specified.

5.     The differencing map (DIFF=) is used to calculate and make an image of the difference between two maps. Ensure rasters are of the same dimension.

6.     Give relative path to geoproject  directory for the map paths

 



The format of the legend files is as follows:

For a categorical map, where there are a number of Category_val’s in the raster, use:

Category_val_1 red_val_1 green_val_1 blue_val_1
Category_val_2 red_val_2 green_val_2 blue_val_2
… and so on.

Where red_val, green_val, blue_val are rgb representation of a colour and take values from 0 to 255.



For a gradient map, where values in the raster range from min_val_for_stretch, to max_val_for_stretch.

transform = <Linear or Log>
red0 = red_val
green0 = green_val
blue0 = blue_val
red1 = red_val
green1 = green_val
blue1 = blue_val
min = min_val_for_stretch
max = max_val_for_stretch.

Where red_val, green_val, blue_val are rgb representation of a colour and take values from 0 to 255, and the colour in which a cell is painted is a linear interpolation between rgb0 to rgb1.
