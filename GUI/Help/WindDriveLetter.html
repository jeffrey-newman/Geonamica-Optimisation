Specifies how the wine drive symlink is created which makes a wine drive pointing to the working geoproject directory.

In the optimisation, a geoproject directory is needed, which contains the project files for the Geonamic simulation. A symlink is created to this folder in wine, within the ‘dosdevices’ folder of the prefix being used. This makes a wine drive pointing straight to the geoproject folder, which makes calling the geoproject simpler.

Format:

1. “choose for me”

The optimisation attempts to pick a drive letter which does not conflict with the setup of the prefix.

2. <Letter>:

Make the drive as specified by <letter> (e.g. specifiying m: would make an ‘m’ drive in the prefix pointing to the Geoporject folder)

Notes:

Do not provide the quotation marks or the <, > demarkation symbols.
